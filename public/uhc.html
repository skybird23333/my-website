The only tool you will ever need for uhcs

Lightweight

 There are no oversized resources at all, only a small behaviour pack which is around 50KB(0.05MB). It runs on its own and does not require any dependencies, ensuring that players can join with lightning speed.

Expandable

 The add-on comes with various options such as time length and border size, so that you can create the perfect UHC game directly fron your dream. It is also compatible with most other addons as it does not edit the in-game bahviours, so feel free to spice it up a little bit.

Optimized

 The built-in optimizing feature clears up garbage entities and maximizes code efficiency for the Bedrock Edition Engine, making sure that game runs at full tickrate so lagging will not be a problem at all. It makes the game run even smoother than vanilla Minecraft.

Flexible

 The pack supports down to 1.13 for both Minecraft:Eedrock Edition and Minecraft: Education Edition. Since it is based on mcfunction and Molang and not scripts, it is compatible with all platforms that run the Minecraft Bedrock engine, especially consoles.
